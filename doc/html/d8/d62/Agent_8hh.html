Detailed Description

Definition of an Agent.

Provides declaration for the Agent class. This is basically a composition of TeleoReactors. The Agent will allocate reactors and connect them together according to their configuratiun requirements.

It will also play the role of middle-man to route observations from sender to receiver.

Note:
The Agent class is not thread safe. If run on a separate thread, it cannot be accessed further.
