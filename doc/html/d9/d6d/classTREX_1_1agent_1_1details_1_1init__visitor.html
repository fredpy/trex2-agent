Detailed Description

Reactors initialization visitor.

This class is a depth first search visitor that will initialize all the reactors from the least dependent to the most dependent in the graph.

The initialization is done through the initialize call on each reactor that will in turn call handleInit callback insisde this reactor

Finish vertex callback.

Parameters:
[in]rA reactor
[in]gThe graph r is attached to

This method will be called during a depth first search when all the reactors r depends on have been already explored.

It will call the initialize method of r with a final tick value as provided by the m_finalTick attribute in order to allow this reactor to complete its initialization.
