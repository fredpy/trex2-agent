XML parsing constructor

Parameters:
nodean XML node

Create a new domain based on the content of node. The type of StringDomain is string.

A full domain is defined by the empty element : 

 <string/>

A string containing the two values "foo" and "bar bar" is described by 

 <string>
   <elem value="foo"/>
   <elem value="bar bar"/>
 </string>
