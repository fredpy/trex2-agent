Detailed Description

template<class AbstractProduct, class Id, class ConsArg, class ProductRef = AbstractProduct *, class IdComp = std::less<Id>>
template<class Product>
class TREX::utils::Factory< AbstractProduct, Id, ConsArg, ProductRef, IdComp >::declare< Product >

Simple producer declaration.

Template Parameters:
Productthe real product type

This class is the simplest implementation for a Factory producer. The production is simply calling the constructor of Product accepting ConsArg as an argument.

It is a simple illustration on how a concrete producer can register and be manipulated by a Factory.

This producer wil be available in the Factory during its full lifetime and will automatically unregister on destruction.

Precondition:
Product has a constructor that accepts ConsArg has an argument
Product derives from AbstractProduct
Product is a concrete class (no pute virtual method)
