Detailed Description

TREX basic utilities.

This namespace embeds different classes that are not necessarily specific to TREX but eases different implementation aspect of it.

You'll find here classes providing very basic features such as thread management, exceptions, and other general design patterns.

Normally classes and code defined here is provided by the dynamic library named TREXutils.
