Detailed Description

Conflicting reactor names.

This mthod is throwned when multiple reactors in a graph have the same name.

Indeed, while reactors are identified internally using their memory address, we also did enforce that it is not possible to have multiples reactors with the same name in a transaction graph. This choice was made to avoid confusion on the different logs produced by TREX.
