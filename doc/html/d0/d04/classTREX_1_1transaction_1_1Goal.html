Detailed Description

TREX goal.

This class represent a goal as exchanged between reactors a goal is a predicate with the 3 temporal varaiables start, duration and end. It is in substance what is called a Token inside the EUROPA planning framework.

This class embeds also suficient code to ensure that the duration is at least 1 TICK and that any modification of one of the temporal attributes is propagated to the others so start+duration=end is maintained

Author:
Frederic Py fpy@m.nosp@m.bari.nosp@m..org

need to be extended to embed representation from 3rd party planners without copy so it improve efficiency.
