Detailed Description

template<class IdTraits, class Cmp = std::less<typename IdTraits::id_type>>
class TREX::utils::id_mapper< IdTraits, Cmp >

Embedded key comparison helper.

Template Parameters:
IdTraitskey extraction traits
Cmpkey comparaison functor

This class is a generalize implementation helper for sorted associative containers. It provide the basic methods to find an object in such container. The generalization is made using :

  • IdTraits to identify how to extract the key from an object
  • Cmp to compare the keys from one to another

While Cmp is a classical C++ sorting functor (std::less is the default template value), IdTraits is the addition that allows to generalize sorted associative container.

An IdTraits is a simpel structure that needs to provide the following static attributes :

  • id_type A typedef that defines the type of the key used
  • base_type The type of the input argument from where the key can be extracted
  • id_type get_id(base_type) the method of extraction of the key

For example the IdTraits fro a std::map can be defined as follow: 

 template<typename Key, typename Val>
 struct map_id_traits {
   typedef std::pair<Key const, Val> base_type;
   typedef Key                       id_type;

   static id_type const &get_id(base_type const &v) {
     return v.first;
   }
 };

The extractor for std::set class is even simpler as it maps the object to itself: 

 template<typename Ty>
 struct set_id_traits {
   typedef Ty const  base_type;
   typedef base_type id_type;

   static id_type const &get_id(base_type const &v) {
     return v;
   }
 };
