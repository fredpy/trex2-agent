Detailed Description

template<typename Ty, class Comp = std::less<Ty>>
class TREX::transaction::IntervalDomain< Ty, Comp >::bound

interval bounds extension of Ty

This class extend the type Ty with the tow specific values -inf and +inf. It is used to represent interval domains.

template<typename Ty, class Comp = std::less<Ty>>
bool TREX::transaction::IntervalDomain< Ty, Comp >::bound::operator== ( bound const &  other) const [inline]

Equality test.

Parameters:
othervalue to compare

Test if current instance is equal to other.

Note:
To ensure genericity this test relies on Comp instead of basic subjacent equality test of Ty. This has two side effects :
  • The test may be inneficient as it relies on the fact that a==b is equivalent to !(a<b || b<a). Whivch may require more computation when used with a simple ordering between basic C types
  • For the reasonds mentionned above one need to make sure that Comp is a complete order
Return values:
trueif *this==other
falseotherwise
