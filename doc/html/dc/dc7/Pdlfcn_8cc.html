Detailed Description

unix dl* wrappers to support different platforms

Author:
Patrick Daley These functions provide a wrapper for unix dlopen(), dlsym(), dlclose(), and dlerror() functions. When compiled for APPLE, these functions emulate the unix function using Apple's API.
Note:
OS-X 10.4 (Tiger) introduced a native Apple implementation of these functions with GCC 4.0
