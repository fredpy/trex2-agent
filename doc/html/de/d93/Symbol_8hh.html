Detailed Description

Symbol class deintion and utilities

This header defines the Symbol class. Symbol is a class based on standard C++ string but managed via a unique instance reference counting pointer ensuring better performances and memory management for constant strings frequenlty used.

Note:
In this latest implementation this class is just some wrapper of boost::flyweight class which ewas kept just toavoid heavy modifications of the client classes.
