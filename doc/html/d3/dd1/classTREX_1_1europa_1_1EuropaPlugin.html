Detailed Description

Europa extension manager.

This base class is used to add europa extensions to T-REX Assembly.

When a new Assembly is created it will look to all the instances of this class currently existing and will call registerComponents for all of these passing itself as an argument.

This mechanism along with the EUROPA macro adaptations to T-REX allow to augment the Assembly with new europa extensions such as :

  • Constraints using TREX_REGISTER_CONSTRAINT
  • Flaw filters using TREX_REGISTER_FLAW_FILTER
  • Flaw handlers using TREX_REGISTER_FLAW_HANDLER
  • Flaw managers using TREX_REGISTER_FLAW_MANAGER
  • New components using TREX_REGISTER_COMPONENT_FACTORY
  • New matches using TREX_REGISTER_MATCH_FINDER

All of these macro have the same behavior as their europa counterpart (which are not prefixed by TREX_) except they specialize to apply to the specific Assembly passed as argument.
