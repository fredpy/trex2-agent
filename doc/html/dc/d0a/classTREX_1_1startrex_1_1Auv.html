Detailed Description

Auv reactor definition.

This class implements a very simple reactor that emulates a light switch. It provides a light timeline that can be either on (Auv.On) or off (Auv.Off)

Auv::Auv ( TREX::transaction::TeleoReactor::xml_arg_type  arg)

XML constructor.

Parameters:
argAn XML node definition

This constructor is called to generate a Auv reactor based on an XML node. The expected XML format is the following: 

 <Auv name="<name>" latency="<int>" lookahead="<int>" state="<bool>"
        verbose="<bool>" />

Where :

  • state is a boolean value indicating which is the initial state of the light timeline (default is true)
  • verbose is a boolean used to indicate whther we should repeat the light state at every tick or not (default is false)
