Detailed Description

Active external predicate.

This class is a functor acting as a predicate that checks if a @ external instance refers to an active timeline relation.

It is used to implement the active_external class, which using a boost filter_iterator on the external class in order to reproduce the past behavior of this same external which used to only iterate through active relations.
