Detailed Description

Interval Domain base class.

This class provide a unified abstract interface for implementing interval domains. It especially provide a way to ease and unify the access methods and input/output for all the domain of this kind.

Author:
Frederic Py fpy@m.nosp@m.bari.nosp@m..org
