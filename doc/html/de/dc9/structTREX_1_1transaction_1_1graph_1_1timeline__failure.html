Detailed Description

Timeline relation creation failure.

This class is used to indicate when a timeline operation -- such as declaring an Internal or External timeline -- in the graph failed. It is used to indicate the nature of the error and will be reported to the reactor that made the request through failed_internal and failed_external callbacks

See also:
TeleoReactor::failed_external
