template<class IdTraits, class Cmp = std::less<typename IdTraits::id_type>>
class TREX::utils::list_set< IdTraits, Cmp >

List based generalized set.

Template Parameters:
IdTraitsA traits that helps element key extraction
CmpA comparison functor

This class is a simple generalized implementation of a set/map using a std::list as the internal container.

It relies on the id_mapper class to locate elements

Note:
This implementation is a simple assovciative container. It implies that it ensure that there's one and only one element with a given key
See also:
class id_mapper
Author:
Frederic Py fpy@m.nosp@m.bari.nosp@m..org

This method Find the position of v in the set and insert it if there is no element with the same key as v
