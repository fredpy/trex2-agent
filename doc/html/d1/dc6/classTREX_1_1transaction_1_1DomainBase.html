Detailed Description

Abstract domain definition.

This class is an abstract representation of a domain as manipulated by TREX. A domain specifies a set of possible values generally for a Variable

Author:

void DomainBase::accept ( DomainVisitor &  visitor) const [virtual]

Visiting method.

Parameters:
visitorA domain visitor

This method implement the Visitor design pattern for TREX domains. It will call visitor methods depending on the actual type of this instance.

There's 3 possible cases supported :

  • this class is a BasicInterval instance
  • this class is a BasicEnumerated instance
  • this class is none of the above

in each case the corresponding method of visitor will be called
