Detailed Description

Graph traversal category.

This pseudo-type is used by the boost graph library in order to know the nature of a reactor graph and how it can be manipulated.

So far we specified that a transaction graph repect the following interfaces/concepts as specified in BGL:

  • Incidence Graph (http://www.boost.org/doc/libs/1_43_0/libs/graph/doc/IncidenceGraph.html)
  • Vertex List Graph (http://www.boost.org/doc/libs/1_43_0/libs/graph/doc/VertexListGraph.html)
  • Adjacency List Graph (http://www.boost.org/doc/libs/1_43_0/libs/graph/doc/AdjacencyGraph.html)

The number of concepts supported may grow in the future as the implementation of the transaction graph between reactors is improved.
