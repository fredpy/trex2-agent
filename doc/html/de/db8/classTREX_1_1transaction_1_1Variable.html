Detailed Description

TREX variable representation.

This class implements a TREX variable. variables are used for representing any attribute of an Observation or Goal excahnged between reactors. They are represented by a symbolic name and its associated domain.

Create a new instance by parsing the content of node. This content should be similar to the following

 </Variable>

Where <name> is a non empty string and domain is an XML domain description supported by DomainBase::xml_factory.
