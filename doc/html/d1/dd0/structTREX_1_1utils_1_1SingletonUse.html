Detailed Description

template<typename Ty>
struct TREX::utils::SingletonUse< Ty >

Singleton accessor.

Template Parameters:
Tytype of the singleton

This class is used to access to a phoenix singleton. It also manages the singleton creation and lifetime through reference counting

template<typename Ty>
TREX::utils::SingletonUse< Ty >::SingletonUse ( )

Constructor.

Create a new instance referring to the singleton Ty. If this singleton does not already exist it is created using the default constructor of Ty. On any case the reference counter for the singleton Ty is incremented by 1.
