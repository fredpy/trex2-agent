Detailed Description

simple utilities library

This library embeds all the basic utilities manipulated by higher level TREX components. They are all provided by the library named lib/libTREXutils

Author:
Frederic Py fpy@m.nosp@m.bari.nosp@m..org

std::string const & TREX::utils::internals::p_dlext ( )

Dynamic library extension helper.

This function helps to identify the usual extension of dynamic libraries for the current platform. Indeed, depending on the OS this extension differ.

The cases that are supported now are:

  • .so for linux and most of the UNIXes
  • .dylib under MacOS X
  • .dll under Windows
Returns:
The usual dynamic library extension for this platform
See also:
void *p_dlopen(const char *, int)
