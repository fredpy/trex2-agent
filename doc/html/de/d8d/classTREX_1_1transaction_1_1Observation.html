Detailed Description

Reactor observation.

This class encapsulate the latest observation on a timeline. Each time a timeline value changes/is updated the reactor owning this timeline is producing tis observation which in turn is dispatched to all the reactors tha observe the corresponding timeline.

See also:
void TeleoReactor::postObservation(Observation const &obs)
void TeleoReactor::notify(Observation const &obs)
Deprecated:
the current implementation of observations is not as flexible and transparent as desired and may change drastically in the future
Author:
Frederic Py fpy@m.nosp@m.bari.nosp@m..org
